# Spirit Viewer

The Spirit Viewer is an interactive tool for exploring all 12 spirits and their evolution animations.

## Features

- **12 Spirits**: View all available spirit characters
- **8 Evolution Levels**: See how each spirit transforms from level 0 to level 7
- **Animation Types**: Watch different animations including:
  - Idle animation
  - Attack animation
  - Side attack animation
  - Movement animation
- **Interactive Controls**:
  - Adjust animation speed (4-30 FPS)
  - Toggle animation on/off
  - Show/hide spirit information
  - Choose between small, medium, or large display sizes
- **Quick Navigation**:
  - Spirit gallery for quick switching
  - Evolution progression viewer
  - Real-time preview with live controls

## How to Use

1. **Select a Spirit**: Use the dropdown or click any spirit in the gallery
2. **Choose Evolution Level**: Slide the level control (0-7) to see transformations
3. **Pick Animation Type**: Click the animation buttons (some may be disabled for lower levels)
4. **Adjust Settings**: Control speed, size, and visibility options
5. **Explore**: Hover over the sprite to see frame-by-frame controls

## Animation Availability

Not all spirits have all animation types at every evolution level:
- **Early levels (0-3)**: May only have 1-2 animation types with fewer frames
- **Mid levels (4-6)**: Typically have 2-3 animation types
- **High levels (7)**: Usually have all 4 animation types with full frames

Disabled animation buttons indicate that particular animation type is not available for the selected spirit and level combination.
