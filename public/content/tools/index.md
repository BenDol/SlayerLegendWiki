# Tools & Calculators

Interactive tools and calculators to help you plan and optimize your gameplay.

## Build Tools

### ⚔️ [Skill Builder](/skill-builder)

Create, share, and manage your skill builds with an interactive simulator that mimics the game's UI.

**Features:**
- Visual skill slot management with drag & drop
- Configurable skill slots (1-10)
- Shareable build URLs
- Export/Import builds as JSON files
- Real-time build statistics
- Element distribution tracking
- Rarity-based visual effects

**Perfect for:**
- Planning skill builds before investing resources
- Sharing builds with friends and community
- Testing different skill combinations
- Comparing element distributions

[Open Skill Builder →](/skill-builder)

## Calculators

Interactive calculators for various game mechanics are available in the Database section:

- **Damage Calculator** - Calculate damage output with different stats
- **Enhancement Calculator** - Calculate upgrade costs for equipment
- **Fusion Calculator** - Plan equipment fusion with resource costs
- **Stat Calculator** - Calculate stat changes after promotion
